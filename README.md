#agrinho2026 #agrinho

O meu projeto para o Agrinho é o "AGRO-LOGISTICA-SIMULATOR-PRO-v1.0-agrinho", um jogo de estratégia e simulação onde o jogador assume o papel de gestor de logística de uma cooperativa agrícola. O objetivo é planejar as rotas dos caminhões, gerenciar a frota e organizar o armazenamento nos silos para entregar as safras com eficiência. O projeto foi desenvolvido pensando diretamente no tema "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente", desafiando o jogador a buscar a alta produtividade ao mesmo tempo em que reduz o impacto ambiental. Para vencer, é preciso escolher caminhos que economizem combustível, diminuam as emissões de poluentes e evitem o desperdício de alimentos, tudo isso enfrentando imprevistos como mudanças climáticas, estradas bloqueadas e flutuações de mercado.

Na parte técnica, o jogo foi desenvolvido do zero utilizando apenas JavaScript Puro, HTML5 C e CSS3. A lógica do simulador funciona através de um estado centralizado, que é uma estrutura que armazena e monitora todas as informações do jogo em tempo real, como o saldo financeiro, os níveis de poluição e o estoque dos silos. Quando o usuário interage com a tela, os eventos nativos do navegador disparam funções que processam as decisões (como o despacho de frotas e cálculo de rotas) e atualizam os gráficos imediatamente através do Canvas. Essa organização deixa o código limpo, leve e bem estruturado.
